I этап. Постановка задачи

Описание задачи

В магазине продаются обои. Наименования, длина и ширина рулона известны. Для удобства обслуживания надо составить программу, которая позволит определить необходимое количество рулонов для оклейки любой комнаты.

Известно, что форма комнаты прямоугольная, размеры комнаты задаются высотой (h), длиной (a) и шириной (b). 15% площади стен комнаты занимают окна и двери, они не входят в площадь оклеивания. 10% площади рулона уходит на обрезки. Нужно купить один рулон про запас.

ЦЕЛЬ МОДЕЛИРОВАНИЯ

Помочь покупателям быстро определять необходимое количество рулонов обоев.

Расчетные данные

Дополним информационную модель в табличной форме математической моделью.

При расчете фактической площади рулона, которая пойдет на оклейку помещения, надо отбросить обрезки. Формула имеет вид

Sр = (1 - Обр) × l × d.

В прямоугольной комнате две стены площадью ah, и две стены площадью bh. При расчете фактической площади стен учитывается неоклеиваемая площадь окон и дверей

Sком = 2 × (a + b) × h * (1 - НП)

Количество рулонов, необходимых для оклейки комнаты, вычисляется по формуле

N = Sком / Sр + 1

Необходимо также учесть, что количество рулонов должно быть целым числом, но не меньшим, чем значение N.

КОМПЬЮТЕРНАЯ МОДЕЛЬ

Для моделирования будем использовать язык моделирования Python.

Для начала введём переменные. 

h = float(input("h: "))
a = float(input("a: "))
b = float(input("b: "))
nonGluedArea = 0.15 # Неоклеиваемая поверхность 15%
cuttings = 0.1 # Обрезки 10%

l = float(input("l: "))
d = float(input("d: "))

Следующим шагом будет запись формул

Sroom = 2 * (a + b) * h * (1 - nonGluedArea) # Площадь комнаты
Sroll = (1 - cuttings) * l * d # Площадь обрезков

Частное в формуле количества рулонов нужно будет округлить вверх. Для этого мы будем использовать функцию из модуля math. Добавим в начало файла строчку

from math import ceil

К формулам допишем N

N = ceil(Sroom / Sroll) + 1

Выведем результат

print(f'Площадь рулона: {"%.3f" % Sroll}, Количество рулонов: {N}')
Вывод программы
h: 2.6
a: 5
b: 3
l: 10.5
d: 0.5
Площадь рулона: 4.725, Количество рулонов: 9

Можно рассчитывать одновременно большее количество образцов рулона, заддадим из как массив массивов.

# Заменим
l = float(input("l: "))
d = float(input("d: "))
# на
rolls = [[10.5, 0.5], [10.5, 0.6]]

# Будем обрабатывать данные через цикл for
for i in rolls: # В переменную i записывается 
                  один из массивов рулонов
    Sroom = 2 * (a + b) * h * (1 - nonGluedArea) 
    Sroll = (1 - cuttings) * i[0] * i[1] # Обращаемся к длине и ширине

    N = ceil(Sroom / Sroll) + 1
    print(f'l: {i[0]}, d: {i[1]}, N: {N}')
