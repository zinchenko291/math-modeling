Математическое моделирование

Стихотворение

I этап. Постановка задачи

Описание задачи

Мальчик учит стихотворение из 40 строк. Чтобы запомнить первую строчку ему понадобилась всего 1 мин. На каждую следующую он тратит на 10% времени больше. Стихотворение держится в памяти нерадивого ученика не дольше трех часов, а до школы бежать 15 минут. Как организовать заучивание стихотворения?

Цель моделирования

Предлагается исследовать ситуацию и предложить реальные способы ускорения процесса заучивания стихотворения.

II этап. Разработка модели

ИНФОРМАЦИОННАЯ МОДЕЛЬ

Анализ объекта моделирования показывает, что эта система состоит из двух объектов: мальчика и стихотворения. Стихотворение, в свою очередь, может быть разбито на простые элементы — строки. Каждый из объектов, входящих в систему, имеет свои параметры. Связь между объектами этой системы заключается во времени заучивания стиха нерадивым учеником. Математическая модель связывает между собой параметры элементов системы.

Время на заучивание i-й строки:

Ti = Ti-1 × (1 + ∆ (%)).

Суммарное время на запоминание отрывка из i строк:

Si = Si-1 + Ti.

КОМПЬЮТЕРНАЯ МОДЕЛЬ

Для моделирования будем использовать язык моделирования Python.

Для начала введём переменные.

learningTime = 60 # Время заучивания в секундах
deltaTime = 1.1 # Увеличение времени в процентах
amountOfStrings = 40 # Всего строк

Пройдёмся в цикле по всем строчкам стихотворения

print('№ строки\tВремя заучивания(сек)\t Суммарное время') # Заголовок таблицы
for i in range(amountOfStrings):
    Ti = learningTime * deltaTime**i # Используем формулы геометрической прогрессии
    Si = (Ti * deltaTime - learningTime) / (deltaTime - 1)
    print(f'{ i + 1 }\t{ "%.2f" % Ti }\t{ "%.2f" % Si }')

Сделаем вывод времени в минутах и секундах при помощи модуля Datetime

# В начало кода вставим
import datetime
# В цикле добавим
TiTime_format = str(datetime.timedelta(seconds = int(Ti)))
SiTime_format = str(datetime.timedelta(seconds = int(Si)))
print(f'{ i + 1 }\t{ TiTime_format }\t{ SiTime_format }')
