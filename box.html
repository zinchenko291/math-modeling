Коробка

I этап. Постановка задачи

Описание задачи

Имеется квадратный лист картона. Из листа по углам вырезают четыре квадрата и склеивают коробку по сторонам вырезов. Какова должна быть сторона вырезаемого квадрата, чтобы коробка имела наибольшую вместимость? Какого размера надо взять лист, чтобы получить из него коробку с заданным максимальным объемом?

ЦЕЛЬ МОДЕЛИРОВАНИЯ

Определить максимальный объём коробки.

Для вывода формул математической модели составим геометрическую модель в виде чертежа с указанием исследуемых характеристик объекта.

Расчетные параметры объекта определяются по формулам:

c = a - 2b — длина стороны дна;
S = c2 — площадь дна;
V = Sb — объём.

Здесь, a — длина стороны картонного листа, b — размер выреза. Первоначальный размер выреза b0 = 0. Последующие размеры выреза определяются по формуле bi+1 = bi + ∆b

КОМПЬЮТЕРНАЯ МОДЕЛЬ

Для моделирования будем использовать язык моделирования Python.

Для начала введём переменные. Их названия будут совпадать с информационной моделью.

a = int(input()) # Получаем и преобразуем данные в числовые значения
deltaB = int(input())

Попробуем проследить, как будет изменятся объём с увеличением размера выреза

b = deltaB # При b = 0, объём в любом случае равен нулю, 
#поэтому сразу переходим к следующему значению
V = 0 # Изначально объём коробки равен 0
for i in range(20): # Для начала попробуем 20 итераций цикла
   c = a - 2 * b
   S = c * c
   print(f"Объём = {S * b}, при b = {b}") # Выводим значения в консоль
   b += deltaB

b = deltaB
V = 0
while True:
c = a - 2 * b
S = c * c
if (S * b) < V: 
   b -= deltaB
   c = a - 2 * b

b = deltaB
V = 0
c = a - 2 * b
while c >= 0: # ограничимся только положительным c
   c = a - 2 * b
   S = c * c
   print(f'Размер выреза = {"%.2f" % b}, ', end='')
   print(f'длина стороны = {"%.2f" % c}, ', end='')
   print(f'площадь = {"%.2f" % S}, ', end='')
   print(f'объём = {"%.2f" % (S * b)}')
   b += deltaB
